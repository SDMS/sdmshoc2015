Edit This Website

Here's your chance to make your mark!

In this activity, you will edit the SDMS Hour of Code website to add your own styles! Use what you learned in "About Me Website" or "Solar System Website" and resources from the internet to "style" (modify) the Hour of Code site. Ms. Koo will choose the best improvements to be added to the live site! (1 change per student).

Ideas:

  • Turn this list into a checklist
  • Make Activity icons change when you hover over them
  • Make Activity icons change after you've clicked on them
  • Give Activity icons rounded corners / different shape
  • Set a wintery background image
  • Change the font and color of the title of the page
  • Add shadows or borders to give the Activity icons a 3-D effect
  • Add animation to the title when you first load the page
  • Add animation to the Activity icons when you first load the page
